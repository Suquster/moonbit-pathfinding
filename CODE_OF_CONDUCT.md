# 贡献者公约 / Contributor Covenant Code of Conduct

> 本文件采用 [Contributor Covenant 2.1 中文版](https://www.contributor-covenant.org/zh-cn/version/2/1/code_of_conduct/)。
> 英文原版见 <https://www.contributor-covenant.org/version/2/1/code_of_conduct/>。
> 适用范围：`moonbit-pathfinding` 项目的所有代码仓库、Issue、PR、讨论区、
> 聊天频道以及任何以本项目名义参与的线上/线下活动。

---

## 我们的承诺

身为社区成员、贡献者及领导者,我们承诺,让每个参与社区的人都不受骚扰,
无论其年龄、体型、可见或不可见的身心障碍、族群、性别认同与表达、经验水平、
教育程度、社经地位、国籍、个人外貌、种族、种姓、肤色、宗教信仰、性向或性倾向为何。

我们承诺以开放、友善、多元、包容、健康的方式互动,共同建设社区。

---

## 我们的准则

有助于创造正向社区环境的行为包括但不限于：

* 对他人展现同理心与善意
* 尊重不同的主张、观点与感受
* 给予和优雅地接受建设性的反馈
* 承担责任并向受我们错误影响的人致歉,并从经验中学习
* 聚焦于对整个社区最有利的事,而不仅仅是对我们自身

不被接受的行为包括但不限于：

* 使用情色化的语言或图像,以及任何形式的性关注或性骚扰
* 捣乱、侮辱或贬损性的评论,以及人身或政治攻击
* 公开或私下的骚扰
* 未经明确许可,公布他人的私人信息,例如住址或电子邮箱
* 其他在专业场合中可被合理认定为不恰当的行为

---

## 强制执行责任

社区领导者有责任阐明并执行我们所认定的可接受行为的准则,并会针对他们认为
不当、威胁、冒犯或有害的任何行为采取适当且公平的纠正措施。

社区领导者有权利和责任删除、编辑或拒绝评论、提交、代码、wiki 编辑、issue
和其他不符合本行为准则的贡献,并会在适当时机解释仲裁决定的原因。


---

## 适用范围

本行为准则适用于所有社区场合,也适用于当个人在公开场合代表社区时的情形。

代表我们社区的情形举例:使用官方电子邮件地址、通过官方社交媒体账号发文,
或在线上或线下活动中作为指定代表出席。

---

## 强制执行

如您目击或遭受滥用、骚扰或其他不被接受的行为,可通过以下渠道向负责执行的社区领导者举报：

- **项目维护者 GitHub**: [@Suquster](https://github.com/Suquster)
- **私密联系渠道**: 如需非公开处理，请优先使用维护者 GitHub 主页列出的公开联系方式联系；不要在公开 issue 中披露敏感个人信息。
- **仓库 Issue**: 对不涉及隐私的信息，可在 <https://github.com/Suquster/moonbit-pathfinding> 创建标签为 `conduct` 的 issue。

所有投诉都会经过迅速且公平的审查和调查。

所有社区领导者都有义务尊重任何事件举报者的隐私与安全。

---

## 强制执行准则

社区领导者将遵循以下社区影响准则,来认定他们所判断违反本行为准则的行为后果:

### 1. 指正

**社区影响**：使用不恰当的言语或其他在社区中被认定为不专业或不受欢迎的行为。

**后果**：由社区领导者发出不公开的书面警告,阐明违规行为的本质,并解释该行为
为何不恰当。该领导者可能会要求当事人公开道歉。

### 2. 警告

**社区影响**：单一或一系列违规事件所造成的违规行为。

**后果**：警告并说明持续性行为的后果。在特定时间内,当事人不得与相关人员互动,
包括主动和违反本行为准则的执行人员互动。这包括避免在社区空间,以及像是社交媒体
的外部渠道中的互动。违反这些条款可能会导致暂时或永久的封禁。

### 3. 暂时封禁

**社区影响**：严重违反社区准则,包括持续性的不当行为。

**后果**：在特定时间内,暂时禁止与社区进行任何形式的互动或公开交流。
在此期间,不允许与相关人员有任何公开或私下的互动,包括主动和违反本行为准则的
执行人员互动。违反这些条款可能会导致永久封禁。

### 4. 永久封禁

**社区影响**：表现出违反社区准则的行为模式,包括持续性的不当行为、
骚扰个人或对某类群体的侵犯或诋毁。

**后果**：永久禁止在社区内进行任何形式的公开互动。
