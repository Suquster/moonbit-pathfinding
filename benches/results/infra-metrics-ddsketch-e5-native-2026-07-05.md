# E5 · DDSketch 分位数 sketch 基准（native，2026-07-05）

- 实现：`src/infra_metrics/ddsketch.mbt`——Masson, Rim & Lee VLDB 2019
  DDSketch：γ=(1+α)/(1-α) 对数桶映射 i=⌈log_γ(x)⌉，桶代表值取几何中点
  2γ^i/(γ+1)，任意分位相对误差 ≤ α（论文定理 1）；O(1) 插入、稀疏有序桶
  二分定位、任意两 sketch 逐桶相加合并且保持误差界；零值单独计数。
- 环境：moon 0.1.20260629，native release，`moon bench -p benches/infra_metrics_bench`。
- 负载：流式 n 个重尾样本（确定性 xorshift），每 500 个样本查一次 p99。
  朴素侧保存全部样本，每次查询复制 + 排序 O(n log n)。

| n | DDSketch（O(1) add + O(桶) 查询） | 朴素排序求分位 | 倍率 |
|---|---|---|---|
| 2000 | 166.77 µs ± 5.29 | 541.15 µs ± 13.62 | 3.2× |
| 8000 | 693.76 µs ± 26.66 | 8.80 ms ± 0.30 | 12.7× |
| 32000 | 2.70 ms ± 0.08 | 151.38 ms ± 3.80 | 56.1× |

- 每次查询朴素侧 O(n log n) 复制排序随流长线性放大——倍率随 n 持续拉开。

## 正确性证据（误差界证明式测试）

`src/infra_metrics/ddsketch_test.mbt`：
- 定向：空 sketch / 零值与负值归零桶 / 单值相对误差 ≤ α / 合并保计数保分位。
- **E5 误差界 PBT 60 迭代**：α ∈ {0.005, 0.01, 0.02, 0.05}、均匀与重尾混合
  分布、n∈[500,2000)，对 q ∈ {0.01,…,0.99,1.0} 逐个对照精确排序分位，
  断言相对误差 ≤ α（论文定理 1 的证明式检验）。
- **合并误差界 PBT 40 迭代**：样本对半分入两个 sketch 后合并，p50/p90/p99
  仍满足 ≤ α。
- 三后端（native/wasm-gc/js）全绿。

## HdrHistogram（批次 2）· 定精度直方图 vs 朴素排序分位

- 实现：`src/infra_metrics/hdr_histogram.mbt`——Gene Tene HdrHistogram
  （hdrhistogram.org）：值域按最高有效位分指数段、段内 2^p 线性子桶，
  record/percentile 纯位运算无浮点对数，固定内存 64×2^p 桶；任意分位
  相对误差 ≤ 2^-p；低值段（< 2^p）逐值精确；同精度合并逐桶相加，
  异精度按代表值降级重记录。
- 负载：流式 n 个 [0,10^9) 均匀样本（确定性 xorshift），每 500 个样本查
  一次 p99.9；朴素侧保存全部 + 每次复制排序。

| n | HdrHistogram（O(1) record + 位运算） | 朴素排序求分位 | 倍率 |
|---|---|---|---|
| 2000 | 47.45 µs ± 1.17 | 533.92 µs ± 8.46 | 11.3× |
| 8000 | 165.54 µs ± 0.61 | 8.12 ms ± 0.09 | 49.1× |
| 32000 | 647.21 µs ± 1.06 | 140.93 ms ± 2.43 | 217.8× |

## HdrHistogram 正确性证据

`src/infra_metrics/hdr_histogram_test.mbt`：
- 定向：空直方图 None / 低值段逐值精确 / 负值归零 / max 精确追踪 /
  同精度合并保计数保误差界。
- **误差界 PBT 60 迭代**：p ∈ {4,7,10}、微秒延迟与 10^9 重尾混合值域、
  n∈[500,2000)，对 pct ∈ {1,…,99.9,100} 对照精确排序分位，断言相对
  误差 ≤ 2^-p（位运算桶映射的证明式检验）。
- 三后端（native/wasm-gc/js）全绿。
